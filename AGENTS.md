# Repository Guidelines

## Communication
- 永远使用简体中文进行对话和思考

## Documentation
- 编写.md文档时，也使用中文
- 正式文档写到项目的docs/目录下

## Code Architecture
- 前端框架使用React

## Python
- 数据类型尽可能全部定义成强类型
- 使用uv作为虚拟环境管理工具
- 必须使用uv，而不是pip、conda、python、python3，包括依赖管理
- 项目的根目录必须保持简洁，只保留必须存在的文件
- main.py内容简洁，只保留必须存在的代码
- uv虚拟环境优先使用全局虚拟环境，如使用项目级虚拟环境，需要优先争得同意

## React / JavaScript / TypeScript / Next.js
- React强制使用v19版本，不允许使用v18及以下的版本
